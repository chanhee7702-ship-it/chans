
# Lottery Number Generator

## Overview

This project is a simple web application that generates and displays a set of random lottery numbers. The goal is to create a visually appealing and easy-to-use tool for users who want to pick random numbers for their lottery tickets.

## Current Plan

### Feature: Lottery Number Generation

*   **Objective:** Create a user interface to generate and display 6 unique random numbers between 1 and 45.
*   **Steps:**
    1.  **HTML (`index.html`):**
        *   Change the page title to "Lottery Number Generator".
        *   Create a container for the application.
        *   Add a heading.
        *   Add a dedicated area to display the generated lottery numbers.
        *   Add a button to trigger the number generation.
    2.  **CSS (`style.css`):**
        *   Style the overall layout, centering the content.
        *   Apply a modern and clean design to the number display and the button.
        *   Use CSS variables for colors.
        *   Add a subtle background texture and drop shadows for depth.
    3.  **JavaScript (`main.js`):**
        *   Add a click event listener to the "Generate" button.
        *   Implement a function to generate an array of 6 unique integers between 1 and 45.
        *   Implement a function to render the generated numbers into the display area.
